Drowsiness Detection System
Tech Stack: Python, OpenCV
• Developed a real-time driver drowsiness detection system using Python and OpenCV, improving road safety
through continuous monitoring.
• Implemented the Eye Aspect Ratio (EAR) algorithm to accurately detect fatigue, achieving reliable detection in
real-time scenarios.
• Integrated webcam-based monitoring with an alert mechanism, reducing response time and enhancing detection
accuracy.
